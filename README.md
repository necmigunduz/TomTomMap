# TomTom Map

A user can click on the map and get the coordinates of this location easily.
![Photo](https://github.com/necmigunduz/smart-map/blob/master/assets/s1.jpeg)

# Technologies Used

- TomTom Map,
- React,
- JavaScript.

# How to run

- `git clone git@github.com:necmigunduz/TomTomMap.git`,
- `cd TomTomMap`,
- `npm install`,
- `npm start`.
